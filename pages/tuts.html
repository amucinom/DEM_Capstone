Make your own DEM Model

Items Needed:

  1. Height Maps or RAW files
  2. Unity Game Engine v5.3.4+
  3. Model Renderer

Steps:

  1. The first thing you need to do is have some sort of height maps and or Raw Files. Height maps are black and white gray scale photos of terrain. A great place to export some is terrain.party which uses USGS Shaded Relief to generate the terrain.
  2. Once you have exported the desired terrain you then will have a folder where you will want to choose the image that ends with “(Merged)” which combines the other images to the perfect balance.
    • Photoshop or another photo editing software will work. In Photoshop import the desired height map or relief map and simply open it, if it looks correct then go to Save As and save as a Photoshop Raw Format.
  3. Now that you have a readable file, you are going to use Unity to create the terrain and render it.
    • Unity is the simplest to use as it is free, has the correct export function, and allows the user to edit the features and make them as realistic as they please. Get Unity here!
    • Create a terrain object, click on the gear icon with the terrain object selected, click on import Raw.., Find your Raw file, choose between the Windows or Mac Option VERY CRITICAL, if the terrain is rendered wrong then choose the other option, Make sure to select FLIP.
    • The X and Z are for how large to render the terrain, I use 200 and 200. The important axis is Y which will determine the height.
    • For the Y axis you would most likely use 25-35 for the best results. If the terrain looks weird or not as in depth, then play with this number.
    • Again in the gear icon under Pixel Density, lower it to 1 for the most detail.
    • At this point discover how to use unity to edit the terrain, add textures if desired, add a water plane, smooth the terrain, have fun with it.
  4. Rendering Software: I simply use Sketchfab, https://sketchfab.com which Is a website where you can upload models and it generates a user interactable 3D rendering of the model which can be embedded to any other site. Also there is a direct Export add on asset for “Unity->Sketchfab”. Download the asset and click on Import. To make sure you have it, in Unity click on WINDOW and you will see the option “Export Selection to Sketchfab”.
  5. These are all the steps to find your own data, import and edit in unity, and share it using a rendering software. In general have fun with it and even use the terrain for creating a game in unity!
